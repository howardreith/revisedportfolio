EREaton.com

Project Info:

EREaton.com is a bootstrap-based website built for automotive chemical engineer Edward Eaton. The client requested the ability to personally update blog entries. For this reason, I set him up with a Wordpress blog template, as it would be simpler to use than teaching HTML modification.

Features:
  • Inter-page transition animations
  • Contact and Subscription forms
  • Text and image animations
  • Integrated Wordpress blog
  • Mobile-first design
  • Created by: Howie Reith
  • Last Revised: 17th October, 2018
  • Skills: HTML5, CSS3, Wordpress
  • Client: Edward Eaton

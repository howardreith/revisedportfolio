The Movie-Finder - WhatMoviesWouldILike.com

Project Info:

The Movie-Finder is a machine learning application that accepts the title of a movie, goes through a database of movie ratings, finds movies that users who rated your favorite movie highly also rated highly, and returns the top 15 results.

Features:
  • Front-end memory-efficient search engine using REGEX.
  • Django API built with Django REST Framework
  • Machine learning algorithm using Numpy and Pandas
  • 100,000 ratings (in version 1), more to come in future

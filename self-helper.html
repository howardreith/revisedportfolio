The Self-Helper

Project Info:

One of the greatest challenges in any self-improvement endeavor is simply following through on the pursuit of one's goals. My research on the topic has indicated that one of the best means of assuring one's own commitment is documenting one's progress. The Self-Helper is a tool to help users achieve that end.

Features:
  • Private user data and protected API authentication.
  • 8 separate PostgreSQL resources
  • Conditional logic handled in the Handlebars Template via #IF(OR)
  • Dynamic workouts logging template that changes according to workout selected
  • Sortable tables that log all entries
  • Local and global navigation bars.
